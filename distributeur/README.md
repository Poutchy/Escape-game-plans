Distributeur de Disquette
==========================

Description
-----------

Le "distributeur de disquette" est une énigme interactive pour escape-game. Cet objet est un distributeur mécanique équipé d'un **cadena à 3 chiffres** qui contrôle l'accès à une disquette stockée à l'intérieur. 

Une fois le bon code à 3 chiffres saisi, le mécanisme se déverrouille et permet au joueur de récupérer la disquette.

Caractéristiques
----------------

- **Cadena numérique** : code à 3 chiffres (000 à 999)
- **Mécanisme de verrouillage** : distribue la disquette lorsque déverrouillé
- **Fabrication** : impressions 3D de récupération (matériel durable et économique)
- **Format** : disquette standard (à placer à l'intérieur du distributeur)

Structure du projet
-------------------

- `code/` : fichiers de code ou scripts (si présents)
- `models/` : modèles 3D pour impression (fichiers .stl, etc.)
- `PDFs/` : notes, schémas et documentation
- `schemes/` : schémas de fonctionnement ou diagrammes d'assemblage

Énigme et Code
-------------------

### Contexte
Pour résoudre cette énigme, il faut utiliser le livre **"Construction logique de programmes COBOL"**.

### Énigme
> Un langage ancien détient la réponse.  
> Cherche le moment de sa naissance.  
> Et souviens-toi : le lieu de la réponse fait partie de la clé.

### Indices
- Le langage mentionné est **COBOL**, un langage de programmation ancien.
- Il faut trouver **l’année de sa création**.
- La **page du livre où se trouve l’information** fait également partie de la réponse.

### Solution
La réponse est : 460


### Explication
- **4** → la page du livre où l’information est trouvée.  
- **60** → l’année **1960**, correspondant à la naissance du langage COBOL.

La clé combine donc **le lieu (la page)** et **le moment (l’année)**.


Utilisation
-----------

1. Placer la disquette dans le distributeur (voir `models/` pour savoir où)
2. Fermer le cadenas et le régler sur un code secret (ex. 312)
3. Les joueurs doivent résoudre l'énigme pour trouver le code
4. Une fois le code saisi, le distributeur se déverrouille
